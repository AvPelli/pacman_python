FruitSelector¶

class FruitSelector.FruitSelector(game_display, game, won_counter)¶
__init__(game_display, game, won_counter)¶

Creates a FruitSelector.

It’s function is to determine which fruits it should draw during the current level. It also knows

when exactly to draw these, and will draw which fruits are already eaten in the bottom right corner of the screen.

Parameters:
  • game_display – The display to which the fruit object is drawn
  • game – The game that holds this FruitSelector, needed for communication
  • won_counter – The current level, needed to know which fruits to draw
